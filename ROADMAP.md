# ROADMAP — план доведения системы до идеала

Сеть Telegram-каналов с ИИ-автоматизацией. Цель — техническое совершенство.

---

## 🔧 Стадия 1: Фундамент и надёжность
**Цель:** отказоустойчивая, поддерживаемая, тестируемая система.

Содержит: аудит, миграция на БД, логирование (Sentry), retry/circuit breaker, health-check, бэкапы, безопасность, pre-commit hooks, типизация, тесты, CI/CD, документация архитектуры.

**Критерий завершения:**
- ✅ Все тесты проходят
- ✅ Sentry показывает ошибки в реальном времени
- ✅ БД работает вместо JSON
- ✅ Pre-commit блокирует плохой код
- ✅ Система переживает падение любого внешнего API

---

## 📈 Стадия 2: Качество и функционал контента
**Цель:** посты максимального профессионального SMM-уровня.

Содержит: расширение источников (NewsAPI/GNews/Reddit/Telethon), семантическая дедупликация (embeddings), парсинг конкурентов, multi-model orchestration, версионирование промптов, AI-критик качества, 15+ рубрик, защита от галлюцинаций, A/B-тесты, premium Telegram-форматирование, продвинутая генерация картинок, контент-план.

**Критерий завершения:**
- ✅ 5+ источников данных работают
- ✅ AI-критик отсекает плохие посты
- ✅ 15+ рубрик с уникальными промптами
- ✅ 0% дубликатов контента
- ✅ Качественные картинки разных типов

---

## 🌐 Стадия 3: Масштабирование и управление сетью
**Цель:** удобная работа с любым количеством каналов из одного места.

Содержит: CLI создания канала, веб-админка модерации (FastAPI + Tailwind / Streamlit), централизованная БД (Supabase), роли модераторов, аналитический дашборд, cross-posting, smart scheduling, Redis-кеш (Upstash), мультиязычность, очередь задач.

**Критерий завершения:**
- ✅ Веб-админка работает
- ✅ Создание нового канала — одна команда
- ✅ Дашборд с метриками доступен
- ✅ Cross-posting функционирует
- ✅ Поддержка минимум 2 языков

---

## 🚀 Стадия 4: Продвинутые возможности
**Цель:** функционал уровня топовых медиа-агентств.

Содержит: виральные форматы (карусели, цитаты, опросы), генерация инфографики, self-improvement на основе ER, web search в реальном времени, Telegram Userbot (Telethon), Vision API анализ конкурентов, авто-комментирование, breaking news, webhook, идеальная UX модератора.

**Критерий завершения:**
- ✅ Виральные форматы работают
- ✅ Self-improvement активен
- ✅ Реакция на горячие новости в реальном времени
- ✅ UX модератора на уровне SaaS-продуктов
